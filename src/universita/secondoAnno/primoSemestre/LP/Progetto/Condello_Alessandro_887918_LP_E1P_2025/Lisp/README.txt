Il codice lisp relizza un URI parser basandosi 
sulle specifiche contenute nella consegna del progetto.

La funzione principale è urilib-parse, 
essa ritornerà una classe di nome urilib-structure.
Sono state implementate delle funzioni ausiliari 
per consentire di analizzare i campi degli uri.
Per estrarre i campi dalla classe urilib-structure 
sono stati implementati i metodi richiesti dalla consegna.
Per stampare i dati dalla classe è stata implementata 
la funzione urilib-display.

E' stato supposto che, affinche la porta fosse impostata a 80 di default, 
almeno l'host deve essere stato inserito.
